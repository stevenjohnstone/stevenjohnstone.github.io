Golang Types and Secrets

2021/01/01

TL;DR

Don’t pass around secrets as strings or byte slices. Make a type for secrets to prevent timing attacks through insecure comparisons and leakage through logs. In addition, using a bespoke type to represent secrets makes it easier to spot that a secret is, well, a secret when reading code.
